# JSLearn

JavaScript learning project

## Установка

```bash
npm install
```

## Запуск

### Запуск тестов

```bash
# Запустить все тесты один раз
npm test

# Запустить тесты в режиме наблюдения (автоматически перезапускаются)
npm run test:watch

# Запустить тесты с подробным выводом
npm run test:verbose

# Запустить интерактивный UI для тестирования (открывается в браузере)
npm run test:ui
```
